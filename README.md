# 项目名称
    react管理系统

# 项目构建
1. npx react-create-app robot_business_manage
2. 引入dva 配置redux、router
3. react-app-rewired 配置webpack
4. window.fetch 调用http请求

# 项目结构
    

# 项目启动
1. npm install 下载包
2. npm start 启动项目
3. npm run build 打包项目

# 项目技术库
1. antd 4x UI库
2. dva 路由和状态管理
3. react-animated-router 页面切换动画
4. react.lazy 基于路由的代码分割
